sass使用方法: 在該page .class{ 這裡放入
@include name();
}

例如： .index h2{ @linclude font_h2(); }

對著名字點兩下 快速選取後複製


(目前多數沒設定margin padding display)

(html包法先不參考此檔，主要看sass方便開發使用及測試css設定正確性)
